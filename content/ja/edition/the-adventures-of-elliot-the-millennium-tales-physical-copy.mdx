<Section icon={Trophy} title="コレクターズエディションの同梱内容">

HD-2D RPGの熱心なファンのために、スクウェア・エニックス e-STORE限定のコレクターズエディションが用意されています。このプレミアムパッケージは、ゲームの中心的なテーマである「時間旅行」と、エリオットの相棒である妖精「フェイ」をモチーフにデザインされています。このエディションは公式ストア限定販売となるため、在庫数は非常に限られており、早めの予約が必要です。

コレクターズエディションの目玉は「**フェイと時の扉 デスクトップクロック**」です。これはエリオットが「安寧の時代」「復興の時代」「魔法の時代」「胎動の時代」を行き来するために使用する古代の時の門を模した、実用可能で非常に精巧なフィギュア型時計です。

| アイテム | 説明 | 用途 |
| | | |
| ゲーム本編（パッケージ版） | 選択したプラットフォームの通常パッケージ版 | ゲーム本編のプレイ |
| デスクトップクロック | フェイと時の扉をテーマにした実用時計 | プレミアムフィギュア型時計 |
| オリジナルサウンドトラック | 厳選されたゲーム内楽曲を収録 | 音楽コレクション |
| デラックスアップグレード用コード | ゲーム内アクセサリーパックの引き換えコード | デジタル特典の追加 |

これらの特典は、主に武器のスロットに装着してステータスや効果を変化させることができる特別な「魔石の欠片」です。

</Callout>

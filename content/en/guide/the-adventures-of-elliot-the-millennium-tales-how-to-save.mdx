<Section icon={Swords} title="Troubleshooting Save Errors and Corrupted Data">

Save data corruption can ruin dozens of hours of exploration. Common issues include power interruptions during autosaving, hard drive write errors, or Steam Cloud synchronization conflicts. Taking preventative measures and knowing how to recover lost files can safeguard your progress.

On PC, files can occasionally be blocked by aggressive antivirus software, preventing the game from creating new save directories. On consoles, system updates or sudden crashes are the primary culprits behind corrupted save files.

<Callout type="warning" title="Backup Recommendation">
PC players should regularly copy their local save files to an external directory to prevent progress loss from unexpected hardware failures or cloud sync errors.
</Callout>

Refer to the troubleshooting guide below to resolve common save-related issues:

| Issue | Potential Cause | Recommended Solution |
| :--- | :--- | :--- |
| **Save file not found** | Profile mismatch or directory change | Verify you are logged into the correct platform profile |
| **Cloud sync conflict** | Local and cloud files do not match | Choose the file with the most recent timestamp |
| **Demo save missing** | Demo deleted before retail launch | Ensure demo save file remains in system local storage |
| **Infinite save loop** | Antivirus blocking directory access | Whitelist the game executable in your security software |

If you are playing on PC via Steam, you can locate your local save files by navigating to your Steam installation directory, opening the `userdata` folder, selecting your unique user ID, and locating the folder matching the game's AppID (`3483510`). Copying this folder to a secure location serves as an excellent manual backup.
